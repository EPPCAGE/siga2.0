Como configurar
Modelo de e-mail
EmailJS não configurado. Preencha as credenciais abaixo para ativar o envio real de e-mails. Sem configuração, as notificações são registradas apenas no log abaixo.
O EmailJS é gratuito e permite até 200 e-mails/mês sem nenhum servidor ou backend. Os e-mails são enviados diretamente do navegador.
Passo 1 — Criar conta
Acesse emailjs.com e crie uma conta gratuita com o seu e-mail institucional.
Passo 2 — Criar um Email Service
No painel do EmailJS, clique em Email Services → Add New Service. Escolha Gmail (ou Outlook, se preferir). Autorize a conta de e-mail que enviará as notificações (recomendado: crie uma conta dedicada como noreply.ep@sefaz.rs.gov.br ou use sua conta institucional). Anote o Service ID gerado (ex: service_abc123).
Passo 3 — Criar um Email Template
Em Email Templates → Create New Template, configure o modelo do e-mail. Veja a aba "Modelo de e-mail" ao lado para o conteúdo sugerido. Anote o Template ID (ex: template_xyz789).
Passo 4 — Obter a Public Key
Em Account → General → Public Key, copie a chave (ex: AbCdEfGhIjKlMnOp).
Passo 5 — Preencher as credenciais
Volte para a aba Credenciais, preencha os três campos e clique em Salvar e ativar. Use o botão Enviar e-mail de teste para confirmar que está funcionando.
Cole este conteúdo no seu template do EmailJS. As variáveis entre {{ }} são preenchidas automaticamente pelo sistema.
